基本思路

  • 在 JavaScript 中，只有以下这几个会被认为是假值 (falsy)：false, null, 0, "", undefined, NaN。
function bouncer(arr) {
    // 用于存储结果
    var result = [];
    for (var i = 0; i < arr.length; i++) {
        if (Boolean(arr[i])) {
            // 如果 Boolean 构造器返回 true，就保存到 result 中
            result.push(arr[i]);
        }
    }
    return result;
}

优化思路 使用数组的filter方法

function bouncer(arr) {
    return arr.filter(function (e) {
        return !!e;
    })
}

高级解法 - 同样是 filter

function bouncer(arr) {
    return arr.filter(Boolean);
}
